  Hướng làm: Embed Superset Dashboard với Multi-Tenant Isolation
                                                                                                                                  
  Tổng quan luồng                                           
                                                                                                                                  
  User login (Frontend)                                     
    → POST /auth/login → JWT (chứa tenant_id + role)
    → POST /auth/dashboard-token → Superset Guest Token (chứa RLS clause)                                                         
    → Frontend nhận iframe embed URL + guest token                                                                                
    → Superset query với auto WHERE tenant_id='X'

  Bước 2: Cấu hình RLS (Row Level Security)                                                                                       
                                                            
  Trong Superset UI, vào Security → Row Level Security → (+). Tạo rule cho mỗi bảng:                                              
                                                            
  ┌─────────────┬───────────────────────────────────────────┐                                                                     
  │    Field    │                   Value                   │
  ├─────────────┼───────────────────────────────────────────┤
  │ Table       │ fact_sales                                │
  ├─────────────┼───────────────────────────────────────────┤
  │ Role        │ TenantViewer                              │
  ├─────────────┼───────────────────────────────────────────┤                                                                     
  │ Clause      │ tenant_id = '{{ current_user_username }}' │
  ├─────────────┼───────────────────────────────────────────┤                                                                     
  │ Filter Type │ Regular                                   │
  └─────────────┴───────────────────────────────────────────┘

  (Làm tương tự cho fact_inventory, fact_purchase, dim_customer, dim_store, dm_sales_summary)                                     
   
  Bước 3: Tạo API endpoint /auth/dashboard-token                                                                                  
                                                            
  Endpoint này đã có trong auth.py (dòng 277). Cần nâng cấp để dùng Superset Embedded SDK thay vì guest token cũ:                 
                                                            
  @router.get('/dashboard-token')                                                                                                 
  def get_dashboard_token(                                  
      authorization: str = Header(...),                                                                                           
      dashboard_id: int = 1,
  ):                                                                                                                              
      """                                                   
      Nhận JWT → tạo Superset Guest Token
      → trả về { dashboard_url, guest_token } để frontend embed
      """                                                                                                                         
      # 1. Verify JWT từ Auth Gateway
      token = authorization[7:]                                                                                                   
      payload = decode_token(token)                                                                                               
   
      # 2. Lấy Superset admin token                                                                                               
      admin_token = get_superset_admin_token()              

      # 3. Tạo guest token với RLS                                                                                                
      if payload.role == 'admin':
          rls_clause = '1=1'  # Admin: không lọc                                                                                  
      else:                                                                                                                       
          rls_clause = f"tenant_id = '{payload.tenant_id}'"
                                                                                                                                  
      guest = requests.post(                                
          f'{SUPERSET_URL}/api/v1/security/guest_token/',
          headers={'Authorization': f'Bearer {admin_token}'},                                                                     
          json={
              'user': {                                                                                                           
                  'username': payload.username,             
                  'first_name': payload.role,
                  'last_name': payload.tenant_id or 'admin',
              },                                                                                                                  
              'resources': [
                  {'type': 'dashboard', 'id': dashboard_id},                                                                      
              ],                                            
              'rls': [
                  {'clause': rls_clause},
              ],
          }
      )
      guest_token = guest.json()['token']
                                                                                                                                  
      # 4. Trả về embed URL
      return {                                                                                                                    
          'dashboard_url': f'{SUPERSET_URL}/superset/dashboard/{dashboard_id}/',
          'guest_token': guest_token,                                                                                             
      }

  @login_required                                           
  def view_dashboard(dashboard_id):
      # Gọi API để lấy guest token
      resp = requests.get(                                                                                                        
          f'{API_BASE_URL}/auth/dashboard-token',
          headers={'Authorization': f'Bearer {session["access_token"]}'},                                                         
          params={'dashboard_id': dashboard_id}             
      )                                                                                                                           
      data = resp.json()
      return render_template('dashboard.html',                                                                                    
          dashboard_url=data['dashboard_url'],              
          guest_token=data['guest_token']
      )
