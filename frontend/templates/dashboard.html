Upload source

Kéo thả file nguồn hoặc chọn thủ công.

Chấp nhận `.xlsx`, `.xls` và `.csv`. Sau khi nạp, lịch sử file gần đây sẽ được cập nhật ngay ở phía dưới để bạn xác minh lại staging.

Hướng dẫn file nguồn

Giữ nhịp đặt tên ổn định để ETL phân loại nhanh và để lúc demo bạn giải thích nguồn dữ liệu không bị khựng.

  1. Doanh thu

    Ví dụ: `BaoCaoDoanhThu_HN_20260101.xlsx`

  2. Tồn kho

    Ví dụ: `QuanLyKho_HCM_20260101.xlsx`

  3. Danh mục

    Dùng `DanhMucSanPham.csv`, `DanhMucKhachHang.csv`, `DanhMucNhaCungCap.csv`.

  4. Nhập hàng

    Dùng `PhieuNhapHang.csv` cho luồng nhập kho và nhà cung cấp.
